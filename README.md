# Foissner Lab – Optimiertes Website-Set

Dieses Paket enthält ein vollständiges, visuell überarbeitetes Website-Set für Foissner Lab.

## Enthalten

### Standalone Seiten
- index.html
- ueber-uns.html
- blog.html
- support.html
- downloads.html
- kontakt.html

### Assets
- assets/css/foissner-lab.css
- assets/js/foissner-lab.js
- assets/icons/*.png

### Elementor Snippets
Im Ordner `elementor-snippets` liegt für jede Unterseite ein HTML-Snippet, das als Ausgangsbasis in Elementor HTML Widgets eingesetzt werden kann.

## Designrichtung
- weicher Weißton auf Schwarz / Anthrazit
- Lime Green für CTAs und Akzente
- klare Typografie
- subtile Scroll-Animationen
- hochwertige, technische Buttons im Stil des gelieferten SAWARO Button Kits
- ruhige, elegante, besser lesbare Cards

## Anwendung

### Variante A – Direkt als Website-Vorschau
Die HTML-Dateien einfach lokal öffnen oder auf einen Hoster laden.

### Variante B – In Elementor nutzen
1. `assets/css/foissner-lab.css` als globale CSS-Basis übernehmen
2. Seiten in WordPress / Elementor anlegen
3. Inhalte aus `elementor-snippets/*.html` als Ausgangspunkt in HTML-Widgets oder Containerstrukturen einfügen
4. Navigation, Formulare und Download-Links final im CMS verbinden

## Hinweis
Die Formulare sind im Paket als visuelle Vorlagen eingebaut. Für produktive Nutzung sollten sie mit Elementor Forms, WPForms oder einem anderen Formularsystem verbunden werden.
